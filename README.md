# Remodel 2026 (Update)

This program uses node and Express to accesses a MongoDB database with materials and locations for a bathroom remodel. 

## Instructions for Build and Use

[Software Demo](Put_Your_Video_Link_Here)

Steps to build and/or run the software:

1. npm start 

Instructions for using the software:

1. There is no front end to this software. It is a server for the database. 
2. GO to: https://remodel2026.onrender.com 
3. The home screem shows that path to follow
4. Go to /locations for the locations to be remodeled.
5. Go to /materials for the materials to be used.
